## How to Upgrade

Follow these steps to update your Ivan Kiryanov instance to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full backup of your documentation files and database.

    ```bash
    tar -czf backup-$(date +%Y%m%d).tar.gz /path/to/ivan-kiryanov
    ```

  </Step>
  <Step title="Update Package" icon="download">

    <CodeGroup tabs="npm,yarn">

    ````bash
    npm install ivan-kiryanov@latest
    ````

    ````bash
    yarn add ivan-kiryanov@latest
    ````

    </CodeGroup>

  </Step>
  <Step title="Run Migrations" icon="settings">

    Execute database migrations if applicable.

    ```bash
    npx ivan-kiryanov migrate
    ```

  </Step>
  <Step title="Restart Service" icon="refresh-cw">

    Restart your documentation server.

    ```bash
    pm2 restart ivan-kiryanov
    ```

  </Step>
</Steps>

<Callout kind="tip">
  Test the upgrade in a staging environment first to avoid disruptions.
</Callout>
