## Overview

Ivan Kiryanov provides a powerful documentation platform that lets you create, organize, and publish project docs with ease. You get a centralized space for all your Markdown and MDX files, complete with search, versioning, and collaboration tools. Whether you manage open-source projects or internal team knowledge bases, this platform streamlines your workflow.

Build structured sites from your content using built-in themes and components. Customize layouts, add interactive elements like tabs and steps, and deploy instantly without managing servers.

Here's a simple code example to add a multi-language snippet to your docs:

<CodeGroup tabs="JavaScript,Python">
```javascript
// Fetch your docs API
const response = await fetch('https://api.example.com/docs', {
  headers: { Authorization: `Bearer ${YOUR_API_KEY}` }
});
const docs = await response.json();
console.log(docs);
```
```python
# Fetch your docs API
import requests
response = requests.get(
    'https://api.example.com/docs',
    headers={'Authorization': f'Bearer {YOUR_API_KEY}'}
)
docs = response.json()
print(docs)
```
</CodeGroup>
