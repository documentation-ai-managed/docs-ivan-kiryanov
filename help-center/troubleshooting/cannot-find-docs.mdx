---
title: Cannot Find Documentation
description: Resolve missing documentation issues.
---

## Symptoms

- Docs page returns 404 error
- Can't see your project docs listed

## Solutions

<Steps>
  <Step title="Check document status">
    ````tip
    Make sure the doc is published and not in draft.
    ````
  </Step>
  <Step title="Search for doc name">
    ````tip
    Use the search bar at the top of the dashboard.
    ````
  </Step>
</Steps>
